# Commands

- `>` ⇒ Open a new terminal
- `+` ⇒ Open a new popup
- `@` ⇒ Open a new REPL
- `$` ⇒ Execute commands in a shell
- `&` ⇒ Write an attachable program to `connect.sh` and detach the client
